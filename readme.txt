Custom 404, creates a 404 page that is set to the default one of your WordPress installation.
You can change almost every aspect of this page, background color, font size, style and so on from the dashboard.

== Description ==

Custom 404, helps you customize your 404.
From the dashboard options you're able to modify the background, the image and the fonts color and style.

This page will show users the last post on your site, so that they may choose to follow one of this instead of leaving your page.

== Installation ==

1. Upload custom404 to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
